# [A Personal Website](https://xulsh8.github.io/)

This repository contains the source code for my personal website.

The website is primarily used to document and share my technical learning notes, research insights, and experiences. It serves as a place to organize knowledge, track progress, and record ideas encountered during study and research.

This website is built upon the excellent open-source personal website template created by [tw93](https://github.com/tw93/tw93.github.io). Special thanks to him for making the project publicly available.
